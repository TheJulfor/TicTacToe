# Крестики-нолики C++
Данный проект - классическая игра "Крестики-нолики" в консольном представлении.

## Особенности игры
- Автоматические сохранения
  - После каждого хода игра автоматически сохраняется в файл.
- Возможность выхода во время игры
  - Благодаря автоматическим сохранениям, игрок может без проблем выйти в главное меню посреди игровой партии и продолжить игру позже.
- Удобный ввод
  - Вместо координат клетки, каждая из них просто пронумерована от 1 до 9 для удобства игрока.
- Защита от случайного ввода
  - Если игрок введет что-то ненужное, то игра не вылетит и попросит заново повторить ввод.
- Точное следование правилам
  - Логика игры следит за тем, чтобы игроки не ходили в уже занятые клетки, а также всегда проверяет победу/ничью.

## Модульность игры
- main.cpp - главное меню игры, запуск игры и правила
- gameLogic.cpp - логика игры (отображение доски, ходы, проверки на победу/ничью)
- saving.cpp - сохранение и загрузка игры
